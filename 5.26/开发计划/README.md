APP介绍：《拾猫记》
你本以为日子会就这样平淡、无趣、重复地过下去，直到那天晚上你回家，在门口捡到一只小猫......
它陪着你完成繁杂又琐碎的日常工作，你拿起许久不看的书，点开音乐播放键，依旧重复着同样的清晨，正午和傍晚。
慢慢地，有什么东西不一样了，家里开始到处散落着它的玩具和零食，你们也越来越像家人。
即使是平淡稀疏的日常，有猫在就不一样。
本产品是一款以养猫为主题的功能软件，用户收养一只流浪猫，通过使用定时、笔记本、阅读、日程等功能获取代币，换取与猫咪互动的道具。用户可以尽情地喂食、抚摸猫咪，与猫咪玩耍，培养感情，根据不同的好感度，猫咪会有不同的互动反馈。
